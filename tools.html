Windows Setup


Chocolatey package search

INSTALL CHOCO COMMANDS:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))


PACKAGES:

amd-

choco install --ignore-checksum amd-ryzen-chipset amd-ryzen-master

thinkpad-

choco install --ignore-checksum lenovo-thinkvantage-system-update

choco install --ignore-checksum nvidia-display-driver
