Windows Setup

Windows:


Chocolatey package search

INSTALL CHOCO COMMANDS:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

ARCHLINUX INSTALLS:

//main

During install:

sudo pacman -Sy archinstall

Official Packages:

sudo pacman -Syu audacity bitwarden filezilla freecad gimp gnome-disk-utility libreoffice-fresh neofetch ntfs-3g obs-studio p7zip python qbittorrent signal-desktop sqlitebrowser thunderbird timeshift veracrypt vim vlc wireshark-qt

AUR packages:

Getting yay:

git clone https://aur.archlinux.org/yay.git

cd yay

makepkg -si

packages:

yay -S drawio github-desktop librewolf-bin nordvpn-bin pycharm-professional qflipper visual-studio-code zoom


System services:

Bluetooth

sudo pacman -Sy bluez bluez-utils

sudo systemctl start bluetooth.service

sudo systemctl enable bluetooth.service

Macos:


Brew package search

Packages (brew.sh)-

Install Brew-

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Casks-

brew install --cask android-file-transfer appcleaner audacity bitwarden db-browser-for-sqlite drawio freecad gimp github librewolf microsoft-excel microsoft-outlook microsoft-powerpoint microsoft-word mos nordvpn obs pycharm qbittorrent qflipper rar rectangle signal vlc wireshark zoom

Shell-

brew install neofetch python@3.12 python-tk@3.12 vim

If Install Unsuccessful-

brew uninstall {package}

brew install (--cask/or no cask) --no-quarantine {package}

To Uninstall-

remove with appcleaner and then >>brew uninstall {package}

Resize encrypted images-

hdiutil resize -size {size/e.g. 5g (5 gigabytes)} {encryped folder location (.dmg)}

This is all subject to change (not definitive)
