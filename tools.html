Different commands I use when setting up an OS

I have to set up os's extremely frequently so these are just a list of commands I use to make life easier.
eventually more utilities will be released here


Windows:


INSTALL CHOCO COMMANDS:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))


PACKAGES:

amd-

choco install --ignore-checksum amd-ryzen-chipset amd-ryzen-master

thinkpad-

choco install lenovo-thinkvantage-system-update

Click to download Lenovo Vantage

general-

choco install --ignore-checksum 7zip bitwarden blender chromium dolphin goggalaxy gzdoom intel-dsa librewolf minecraft-launcher nordvpn nvidia-display-driver obs-studio pcsx2 qbittorrent signal steam

disable fast startup


ARCHLINUX INSTALLS:

//main

During install:

sudo pacman -Sy archinstall

Official Packages:

sudo pacman -Syu

AUR packages:

Getting yay:

git clone https://aur.archlinux.org/yay.git

cd yay

makepkg -si

packages:

yay -S drawio github-desktop librewolf-bin nordvpn-bin pycharm-professional qflipper visual-studio-code zoom


System services:

Bluetooth

sudo pacman -Sy bluez bluez-utils

sudo systemctl start bluetooth.service

sudo systemctl enable bluetooth.service
